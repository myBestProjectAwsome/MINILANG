# Contributing to MiniLang

Merci de votre intérêt pour contribuer à MiniLang ! 🎉

## Comment contribuer

1. **Fork** le projet
2. **Créez** une branche : `git checkout -b feature/ma-fonctionnalite`
3. **Commitez** vos changements : `git commit -m 'Add: nouvelle fonctionnalité'`
4. **Pushez** vers la branche : `git push origin feature/ma-fonctionnalite`
5. **Ouvrez** une Pull Request

## Bonnes premières issues

- Ajouter le support des boucles `for`
- Implémenter le type booléen
- Améliorer les messages d'erreur
- Écrire plus de tests

## Code de conduite

Soyez respectueux et constructif dans vos échanges.

Merci ! 🚀
